How to Reduce Picture

Follow the steps below to reduce the size of your picture:

  1. Click on the "Choose File" button to upload your picture.
  2. Enter the percentage by which you want to reduce the picture size.
  3. Click on the "Process Now" button to resize the picture.
  4. Preview the resized picture and download it if you are satisfied with the result.

This tool uses JavaScript to read the uploaded image, resize it using a canvas element, and then provide a download link for the resized image.

JavaScript Code Explanation

The JavaScript code performs the following tasks:

  1. Handles the file upload and previews the uploaded image.
  2. Processes the image by resizing it using a canvas element.
  3. Displays the resized image and provides a download link.
